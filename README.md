## 🚀 스터디 진행 방식
1. 매주 정해진 분량만큼 각자 책을 읽는다.(회차별 읽을 분량에 대해서는 `Issue`에 업데이트) -> 매주 2장씩
2. 스터디 시작 전까지 각자의 생각을 정리(마크다운 형식)하여 레포에 업로드한다. -> 책의 내용, 느낀점, 팀원들과 이야기해보고 싶은점
3. 함께 모여 생각을 공유하고 토론한다.
4. 도출된 결론은 정리하여 레포에 업로드한다.
